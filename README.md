# GameCenter - PC Games Discovery App

A high-performance Android application designed for gamers to discover free PC games in real-time. The app integrates with external REST APIs to provide an up-to-date catalog of gaming deals and offers.

## 🚀 Key Features
* **Real-time API Integration:** Fetches live data from gaming APIs to display the latest free-to-play titles.
* **Advanced Data Management:** Implemented robust **Search, Sort, and Filter** functionalities for seamless navigation.
* **Personalized Experience:** Features a **Favorites** system allowing users to save and track specific games.
* **Rich UI/UX:** Detailed game views including descriptions, metadata, and direct links to store pages, built with a modern and responsive design.

## 🛠 Tech Stack
* **Language:** Java
* **IDE:** Android Studio
* **Architecture:** Modular design using **Android Fragments** and Navigation Graph.
* **Backend/Data:** REST API integration and **Firebase** for user-specific data (Favorites/Auth).
* **UI Components:** RecyclerView for efficient list rendering and custom layouts.
