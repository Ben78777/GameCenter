# 🎮 GameCenter App

A modern Android application designed for gamers to explore, filter, and manage their favorite games. This project was developed as part of a Software Engineering course, showcasing integration with REST APIs and Firebase.

## 🚀 Features
* **Live Game Feed:** Fetches real-time data from the `FreeToGame API`.
* **Category Filtering:** Uses Android Material Chips to filter games by genre or platform.
* **Firebase Integration:** * User Authentication (Login/Register).
    * Real-time Database to save and sync "Favorite" games.
* **Modern UI/UX:** Built with a Fragment-based architecture and Smooth Navigation.
* **Responsive Design:** Optimized layouts using `ConstraintLayout`.

## 🛠 Tech Stack
* **Language:** Java
* **Architecture:** Single Activity with Multiple Fragments.
* **Networking:** Retrofit 2 & Gson for API calls.
* **Backend:** Firebase (Auth & Real-time Database).
* **Navigation:** Jetpack Navigation Component (Navigation Graph).
* **UI Components:** RecyclerView, Material Design 3, Glide (for image loading).

## ⚙️ Setup & Installation
1. Clone the repository: 
   `git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git`
2. Open the project in **Android Studio**.
3. Ensure you have a `google-services.json` file in the `app/` directory (for Firebase).
4. Sync Gradle and Run the app on an Emulator or Physical device.
